# 🌐 Personal Portfolio — Shreya Pal

A modern, responsive personal portfolio website designed to showcase my projects, skills, and journey as a developer.  
Built with a focus on **clean design, smooth navigation, and strong storytelling**.

---

## ✨ Live Demo

🔗 https://playernightqueen16.github.io/Personal-Portfolio/

---

## 🎯 Purpose

This portfolio serves as a **central hub** for:
- Showcasing technical projects and experiments  
- Highlighting skills and areas of interest  
- Presenting achievements, hackathons, and experiences  
- Creating a strong digital presence for recruiters and collaborators  

---

## 🚀 Features

- 🧑‍💻 **Project Showcase**
  - Structured display of key projects with descriptions  
  - Easy navigation across different works  

- 🎨 **Modern UI/UX**
  - Clean layout with intuitive flow  
  - Smooth scrolling and section transitions  

- 📱 **Responsive Design**
  - Optimized for desktop, tablet, and mobile devices  

- ⚡ **Fast & Lightweight**
  - No heavy frameworks — quick load times and smooth performance  

- 🧭 **Simple Navigation**
  - Clear sections for projects, skills, and contact  

---

## 🛠️ Tech Stack

- **HTML5** — Structure and semantic layout  
- **CSS3** — Styling, layout, responsiveness  
- **JavaScript** — Interactivity and dynamic behavior  
- **GitHub Pages** — Deployment

## ⚙️ How to Run Locally

1. Clone the repository
```bash
git clone https://github.com/PLayerNightQueen16/Personal-Portfolio.git
```

2. Navigate to the folder
```
cd Personal-Portfolio
```

3. Open in browser
```
Double-click index.html
```
