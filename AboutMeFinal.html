My Projects

Throughout my journey in web development, I have built several foundational projects to strengthen my understanding of HTML and CSS. My first project, My First Webpage, served as a stepping stone where I learned to structure a complete webpage with headings, paragraphs, links, and images. Following that, I created a Weekend Plans List, a simple yet functional webpage to organize tasks and events, and a Movie Collection Table, which enhanced my skills in creating structured content using tables. The Contact Form project introduced me to interactive elements of a webpage, while the Semantic Webpage helped me practice using semantic HTML tags to improve accessibility and SEO. Additionally, the Recipe Page allowed me to experiment with images, lists, and content organization to present recipes clearly and attractively.


To further explore CSS and design principles, I created projects that focused on layout, styling, and responsiveness. In CSS Shapes, I practiced creating geometric shapes and animations purely using CSS, while Responsive Profile Card taught me the importance of designing for different screen sizes and devices. Grid Layout was a project dedicated to mastering CSS Grid for creating clean, structured, and visually appealing layouts. Similarly, Photo Gallery combined both grid and flexbox techniques to arrange images elegantly while maintaining responsiveness. These projects reinforced my understanding of design principles, spacing, alignment, and visual aesthetics.


Finally, I ventured into more creative and visually dynamic projects, combining HTML, CSS, and animation skills. Moon Orbit and Ferris Wheel allowed me to simulate motion and rotation using CSS animations, pushing my skills in transitions, keyframes, and dynamic styling. Each of these projects, whether functional, aesthetic, or interactive, contributed to building a solid foundation in web development, allowing me to experiment, learn from challenges, and gradually enhance my coding and design capabilities. Collectively, these experiences have shaped my proficiency in HTML and CSS, preparing me for more advanced projects in the future.
